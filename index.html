Introduction

Javascript Guide

Welcome!

This guide will help you write good, maintainable Javascript code. It contains information that will help you take decisions about tooling, runtime selection, etc. quickly.

Basically, the sole purpose of this guide is to give straightforward solutions to many problems that you will face as a Javascript developer. That being said, it is also opinionated, so you might find that some parts of the book are not to your liking. In that case, you should open a pull request.

Also, make use of the search bar!
